Weather App Project

I learned how to use javascript with this app. I learned how to incorporate API into code. I was able to organize code in a simple and easy to use weather app. This app will give the current weather for any city you type. It will also generate a future weather forcast along with images to display the expected weather. It also provides the date and time for any city searched for.
